# 2. Теоретическая часть

## 2.1. Режимы работы редактора vi

Редактор vi имеет три режима работы:

- **Командный режим** — предназначен для ввода команд редактирования и навигации по редактируемому файлу;
- **Режим вставки** — предназначен для ввода содержания редактируемого файла;
- **Режим последней (или командной) строки** — используется для записи изменений в файл и выхода из редактора.

## 2.2. Основные команды редактора vi

### Команды управления курсором

| Клавиша | Действие |
|---------|----------|
| `h` или `←` | Курсор влево |
| `l` или `→` | Курсор вправо |
| `j` или `↓` | Курсор вниз |
| `k` или `↑` | Курсор вверх |

### Команды позиционирования

| Команда | Действие |
|---------|----------|
| `0` | Переход в начало строки |
| `$` | Переход в конец строки |
| `G` | Переход в конец файла |
| `nG` | Переход на строку с номером n |

### Команды редактирования

| Команда | Действие |
|---------|----------|
| `i` | Вставка текста перед курсором |
| `a` | Вставка текста после курсора |
| `o` | Вставка строки под курсором |
| `O` | Вставка строки над курсором |
| `x` | Удаление одного символа |
| `dw` | Удаление одного слова |
| `dd` | Удаление одной строки |
| `u` | Отмена последнего действия |

### Команды режима последней строки

| Команда | Действие |
|---------|----------|
| `:w` | Записать файл |
| `:q` | Выйти из редактора |
| `:wq` | Записать файл и выйти |
| `:q!` | Выйти без сохранения |

# 5. Ошибки и их решение

| № | Ошибка | Причина | Решение |
|---|--------|---------|---------|
| 1 | `LOCAL HELLO=World` не работает | `LOCAL` должно быть строчными буквами | Использовать `local HELLO=World` |
| 2 | Переменная `HELL` вместо `HELLO` | Опечатка в исходном коде | Изменить на `HELLO=Hello` |
| 3 | Не удаётся выполнить команду в vi | Нахождение в режиме вставки | Нажать `Esc` для перехода в командный режим |
| 4 | `q!` вместо `wq` | Желание выйти без сохранения | Использовать `wq` для сохранения и выхода |

# 7. Ответы на контрольные вопросы

## 7.1. Дайте краткую характеристику режимам работы редактора vi

- **Командный режим** — режим, в котором клавиши интерпретируются как команды редактирования и навигации. Переход из любого режима осуществляется клавишей `Esc`.
- **Режим вставки** — режим, в котором вводимый текст добавляется в файл. Переход осуществляется командами `i`, `a`, `o`, `O`.
- **Режим последней строки** — режим для ввода команд, связанных с файлом (сохранение, выход, поиск, замена). Переход осуществляется командой `:`.

## 7.2. Как выйти из редактора, не сохраняя произведённые изменения?

Необходимо:
1. Нажать `Esc` для перехода в командный режим
2. Нажать `:` для перехода в режим последней строки
3. Ввести `q!` и нажать `Enter`

## 7.3. Назовите и дайте краткую характеристику командам позиционирования

| Команда | Характеристика |
|---------|----------------|
| `0` | Перемещение курсора в начало текущей строки |
| `$` | Перемещение курсора в конец текущей строки |
| `G` | Перемещение курсора в конец файла |
| `nG` | Перемещение курсора на строку с номером n |

## 7.4. Что для редактора vi является словом?

Словом в vi считается последовательность символов, ограниченная пробелами, знаками табуляции, знаками препинания или концом строки.

## 7.5. Каким образом из любого места редактируемого файла перейти в начало (конец) файла?

- В начало файла: нажать `1G` или `gg`
- В конец файла: нажать `G`

## 7.6. Назовите и дайте краткую характеристику основным группам команд редактирования

- **Команды вставки**: `i` (перед курсором), `a` (после курсора), `o` (строка снизу), `O` (строка сверху)
- **Команды удаления**: `x` (символ), `dw` (слово), `dd` (строка)
- **Команды отмены**: `u` (отмена последнего действия)

## 7.7. Необходимо заполнить строку символами $. Каковы ваши действия?

1. Нажать `Esc` для перехода в командный режим
2. Нажать `0` для перехода в начало строки
3. Нажать `i` для входа в режим вставки
4. Вводить символы `$`
5. Нажать `Esc` для возврата в командный режим

## 7.8. Как отменить некорректное действие, связанное с процессом редактирования?

Нажать `u` в командном режиме. Это отменит последнее выполненное действие.

## 7.9. Назовите и дайте характеристику основным группам команд режима последней строки

- **Команды работы с файлами**: `:w` (запись), `:q` (выход), `:wq` (запись и выход)
- **Команды настройки**: `:set nu` (нумерация строк), `:set ic` (игнорирование регистра)

## 7.10. Как определить, не перемещая курсора, позицию, в которой заканчивается строка?

В командном режиме нажать `$`. Курсор переместится в конец строки.

## 7.11. Выполните анализ опций редактора vi

Опции редактора vi задаются командой `:set`:
- `:set all` — вывод полного списка опций
- `:set nu` — вывод номеров строк
- `:set list` — вывод невидимых символов
- `:set ic` — игнорирование регистра при поиске

Для отключения опции используется `no` перед именем: `:set nonu`

## 7.12. Как определить режим работы редактора vi?

- **Командный режим**: внизу экрана нет специальных индикаторов
- **Режим вставки**: внизу экрана отображается `-- INSERT --`
- **Режим последней строки**: внизу экрана появляется двоеточие `:` после нажатия `:`
