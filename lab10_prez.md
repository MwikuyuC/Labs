## Что такое vi?

- Экранный редактор Visual display editor
- Установлен по умолчанию в большинстве дистрибутивов Linux
- Три режима работы

---

## Три режима работы vi

| Режим | Назначение |
|-------|------------|
| Командный | Навигация, удаление, копирование, вставка |
| Вставки | Ввод и редактирование текста |
| Последней строки | Сохранение, выход, поиск |

---

## Переключение между режимами

```
                    ┌─────────────────────┐
                    │    КОМАНДНЫЙ РЕЖИМ    │
                    └──────────┬──────────┘
                               │
                Esc ───────────┼──────────── i, a, o, O
                               │
                               ▼
                    ┌─────────────────────┐
                    │     РЕЖИМ ВСТАВКИ     │
                    └─────────────────────┘

                    ┌─────────────────────┐
                    │    КОМАНДНЫЙ РЕЖИМ    │
                    └──────────┬──────────┘
                               │
                Esc ───────────┼──────────── :
                               │
                               ▼
                    ┌─────────────────────┐
                    │  РЕЖИМ ПОСЛЕДНЕЙ СТРОКИ │
                    └─────────────────────┘
```

---

## Управление курсором

| Клавиша | Действие |
|---------|----------|
| h или ← | Влево |
| l или → | Вправо |
| j или ↓ | Вниз |
| k или ↑ | Вверх |

---

## Команды позиционирования

| Команда | Действие |
|---------|----------|
| 0 | В начало строки |
| $ | В конец строки |
| G | В конец файла |
| nG | На строку n |

---

## Команды редактирования

| Команда | Действие |
|---------|----------|
| i | Вставка перед курсором |
| a | Вставка после курсора |
| o | Новая строка снизу |
| O | Новая строка сверху |
| x | Удалить символ |
| dw | Удалить слово |
| dd | Удалить строку |
| u | Отменить действие |

---

## Команды режима последней строки

| Команда | Действие |
|---------|----------|
| :w | Сохранить файл |
| :q | Выйти |
| :wq | Сохранить и выйти |
| :q! | Выйти без сохранения |

---

## Создание файла

```bash
vi hello.sh
```

Нажмите i для входа в режим вставки

Введите текст:

```bash
#!/bin/bash
HELL=Hello
function hello {
    local HELLO=World
    echo $HELLO
}
echo $HELLO
hello
```

---

## Сохранение и выход

1. Нажмите Esc (командный режим)
2. Нажмите : (режим последней строки)
3. Введите wq
4. Нажмите Enter

## Редактирование: изменить HELL на HELLO

1. Установите курсор на HELL
2. Нажмите i
3. Введите LLO
4. Нажмите Esc

## Редактирование: исправить LOCAL на local

1. Установите курсор на LOCAL
2. Нажмите dw (удалить слово)
3. Нажмите i
4. Введите local
5. Нажмите Esc

![После исправления](S10.png)

## Добавить строку в конец файла

1. Нажмите G (перейти в конец)
2. Нажмите o (новая строка снизу)
3. Введите echo $HELLO
4. Нажмите Esc

## Удалить последнюю строку

1. Нажмите G (перейти на последнюю строку)
2. Нажмите dd (удалить строку)

## Сводка команд vi

| Режим | Команда | Действие |
|-------|---------|----------|
| Командный | h/j/k/l | Перемещение курсора |
| Командный | G | В конец файла |
| Командный | dd | Удалить строку |
| Командный | dw | Удалить слово |
| Командный | u | Отменить действие |
| Вставки | i/a/o/O | Вход в режим вставки |
| Последней строки | :wq | Сохранить и выйти |

- Всегда нажимайте Esc перед сменой режимов
- vi различает регистр букв
- Практика помогает освоить vi
